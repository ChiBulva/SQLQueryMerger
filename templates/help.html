How to Use the RFID Query App

Overview

This application lets you view, query, merge, and export data from your local etl_layer.db SQLite database. You can also run custom SQL queries, manage connections, and download results in Excel or CSV format.

  • Click any table name on the SQLite Viewer page to view its contents with fast pagination and search.
  • Use the search box to filter rows by value.
  • Navigate pages using the Previous/Next buttons.

  • Download table or merged table data as Excel or CSV using the buttons next to each table on the SQLite Viewer page.
  • For large tables, prefer CSV export for best performance and file size.

  • Use the Merge Queries or Merge SQLite Tables features to combine tables or queries by matching fields.
  • After merging, the new table appears under Merged Tables on the SQLite Viewer page and can be viewed/exported like any other table.

  • Go to the New Query page to run or save custom SQL queries.
  • Saved queries are listed by connection on the Home page and can be managed from there.
  • Results can be exported or merged with other tables.

  • For large exports, prefer CSV over Excel for speed and file size.
  • Use the Delete button on the SQLite Viewer page to remove tables you no longer need.
For more help, contact your administrator or see the README.
